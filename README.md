## Project Hello World
this project use nodeJs [Express, Mocha, Chai] and deploy on Heroku.

## Requiment
- unit Testing use Mocha & Chai
- Express Deploy on Heroku

## Set up
##### Install Mocha
- npm install mocha --save

##### Install Chai
- npm install chai --save

##### Install Express
- npm install express --save

## Run
##### unit test
- npm test

##### start
- npm start
